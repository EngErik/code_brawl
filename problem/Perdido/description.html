Perdido

Marília tem uma lista de ids dos alunos que acessaram o site e foram contactados pelo time de vendas. Ela quer conferir os dados de alguns dos alunos que não estão na lista para entender melhor a eficiência do seu time.

Ela sabe que os ids são números inteiros sequenciais, e que um id maior que outro indica que ele acessou o site depois. Ela ficou interessada em saber qual é o aluno mais antigo (menor id) que não foi contactado pelo seu time, mas que fez a conta depois de alguém que já foi contactado.

Tarefa

Sua tarefa é escrever um programa que leia a lista de ids e informe qual o id do aluno que a Marília está procurando.

Entrada

Para cada teste, a entrada começa com um inteiro N (sendo 1 <= N <= 103) indicando quantos elementos tem na lista da Marília. Seguem N linhas, cada uma com um inteiro representando um id contactado.

Cada id é um número inteiro de módulo menor que 109.

Saída

Para cada teste, seu programa deve imprimir um único inteiro, o menor id que não está na lista da Marília mas que é maior que algum elemento na lista dela.
