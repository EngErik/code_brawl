QueroIndicar

Lucas teve uma ideia sensacional para aumentar o marketing do QueroBolsa, premiar os alunos que mais indicarem o site para seus amigos. Para isso ele bolou um sistema em que um aluno ganha um ponto se fizer login no site, e ganha cinco pontos se adquirir um cupom. Além disso, quando ele se cadastra ele diz quem foi que o indicou ao site, e quem indicou também ganha pontos.

Entretanto, Lucas achou que seria mais justo se a pessoa só ganhasse pontos das pessoas que ela indicou diretamente ao site. Ou seja se pessoa A indicou pessoa B, e pessoa B indicou pessoa C, a pessoa A não ganha pontos pela pessoa C.

Agora Lucas tem uma lista de todos os alunos que se matricularam enquanto sua campanha de marketing estava ativa e precisa da sua ajuda para determinar quem está ganhando nos pontos de indicação.

Tarefa

Sua tarefa é escrever um programa recebe a lista de alunos - contendo os nome do aluno, quem o indicou, se ele fez login no site e se ele adquiriu um cupom no site - e gerar um placar com a pontuação de cada aluno ordenado de quem tem mais pontos para quem tem menos pontos. Em caso de empate, ordenar alfabeticamente pelo nome do aluno.

Entrada

Em cada caso de teste, a primeira linha da entrada contém um único inteiro N indicando quantos alunos serão descritos. Seguem então N linhas contendo o nome do aluno (que será uma string alfanumérica de até 30 caracteres), o nome da pessoa que o indicou (ou google caso ele não tenha sido indicado), e dois valores booleanos (true ou false) indicando respectivamente se ele tem login e se ele tem cupom.

Não haverá dois alunos com mesmo nome, um aluno com nome google, true ou false. Nenhum aluno é indicado por ele mesmo. Haverá no máximo 104 alunos.

Saída

Para cada teste, seu programa deve imprimir N linhas, uma para cada aluno. Ordenados de quem fez mais pontos para quem fez menos pontos. Cada linha deve ter a posição em que o aluno ficou, o nome do aluno, e quantos pontos ele fez. Seguindo o formato do exemplo.

Exemplos

Entrada Saída 
5
alice google true true
bob google false false
charlie bob false true
denise bob false true
eve denise true false
