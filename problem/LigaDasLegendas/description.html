LigaDasLegendas

João gosta muito de assistir os jogos da LigaDasLegendas em que os times competem para ver quem consegue traduzir um filme mais rápido, e com melhor precisão. Os times são julgados pela ortografia, pela correção das frases, pela sincronia do vídeo com o momento em que as legendas aparecem, dentre outros critérios altamente técnicos.

Na LigaDasLegendas os times competem um contra um, usando o mesmo filme para ser mais justo, e um comitê imparcial decide a pontuação de cada time em cada um dos critérios, e há um único vencedor para cada competição.

João é um torcedor fanático pelo time InTZ, mas acha que eles estão ficando preguiçosos, e como maneira de provar isso, ele quer comparar o desempenho atual deles com o desempenho da melhor época deles. Para isso ele pediu sua ajuda para analisar o histórico de todas as partidas do time e determinar qual o ponto onde o número de vitórias menos o número de derrotas desde o começo do campeonato até esse ponto é o maior possível.

Tarefa

Sua tarefa é escrever um programa para ler o histórico de jogos que o João guardou e determine o número do jogo no qual ao final dele, o saldo de vitórias é o maior possível. Caso haja empate, escolha o mais recente, ou seja, o que contém mais jogos.

Entrada

Para cada teste, a entrada começa com uma linha com um inteiro N (sendo 1 <= N <= 105). Seguem então N linhas cada uma com a string ganhou ou perdeu indicando se o time InTZ ganhou ou perdeu. Os jogos são informados na ordem em que eles aconteceram. Considere que antes do jogo 1 o número de vitórias e derrotas é zero.

Saída

Para cada teste, seu programa deve imprimir uma única linha com 3 inteiros: o número do jogo que o João quer saber, a quantidade de vitórias antes desse jogo, e a quantidade de derrotas antes desse jogo (contando com o próprio jogo em questão).
